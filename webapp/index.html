리얼앱(RealApp)

리얼앱은 (주)우리테크인터네셔날에서 제공하는 부가기능 앱의 명칭입니다.

목록

  • RealGrid Generator
    리얼그리드 컬럼정보를 생성을 위한 구글 스프레드시트용 부가기능입니다.
  • RealApp for Excel
    리얼그리드 컬럼정보를 생성을 위한 엑셀용 부가기능입니다. * Microsoft Office 2013 이상 지원

RealGrid Generator

설치방법

  1. 다음 링크로 이동합니다.
    https://chrome.google.com/webstore/detail/realgrid-generator/kopabannkmogngoabdpipkgoeaclneha
  2. 무료 다운로드를 클릭합니다.
  3. 설치하려면 권한이 필요하므로 아래과 같이 다이얼로그에서 계속을 클릭합니다.
  4. 권한 부여를 위해서 구글 로그인이 되어야 합니다.
  5. 다음과 같은 경고가 표시되지만 이는 정책상의 문제로 불안전한 코드나 바이러스등과 관계없으니 계속 진행합니다.
  6. 설치하려면 권한이 필요하므로 아래과 같이 다이얼로그에서 계속을 클릭합니다.
  7. RealGrid Generator앱 에 SpreadSheet의 읽기, 쓰기 및 Sidebar등의 뷰를 생성할 수 있는 권한을 부여할지 물어봅니다. 허용버튼을 클릭합니다.
  8. 설치하려면 권한이 필요하므로 아래과 같이 다이얼로그에서 계속을 클릭합니다.
  9. 모든 작업이 끝나면 부가기능(Add-ons) - RealGrid Generator - Column Generator 메뉴를 클릭하면 다음과 같이 우측에 사이드바가 표시됩니다.

사용방법

  1. 시트양식화 버튼을 클릭하면 컬럼의 깊이를 입력합니다. 이는 컬럼 그룹핑에 관한 것으로 1이면 그룹이 없은 일반 컬럼, 2이면 상위 그룹이 1계층있는 형태입니다.
  2. 초기에 양식화를 한 후 이후 컬럼 형태에 대해서는 자유롭게 수정할 수 있습니다. 가로로 병합된 컬럼은 그룹으로, 병합되지 않은 컬럼은 일반 컬럼으로 인식됩니다. 단 하나의 행에 수직으로 여러 컬럼이 배치되는 구조는 구현이 불가능합니다.
  3. 컬럼형태가 완성이 되면 필드/컬럼 생성버튼을 클릭하면 아래쪽 필드정보, 컬럼정보 텍스트상자에 값이 채워집니다. 이 상태에서 코드를 복사하여 사용하거나 미리보기 버튼을 클릭하여 다이얼로그 하단의 전체 소스를 복사하여 사용할 수 있습니다.

  4. 엑셀을 실행하여 메뉴에서 RealApps탭이 추가된것을 확인하세요.

사용방법

  1. 시트양식화 버튼을 클릭하면 컬럼의 깊이를 입력합니다. 이는 컬럼 그룹핑에 관한 것으로 1이면 그룹이 없은 일반 컬럼, 2이면 상위 그룹이 1계층있는 형태입니다.
  2. 초기에 양식화를 한 후 이후 컬럼 형태에 대해서는 자유롭게 수정할 수 있습니다. 가로로 병합된 컬럼은 그룹으로, 병합되지 않은 컬럼은 일반 컬럼으로 인식됩니다. 단 하나의 행에 수직으로 여러 컬럼이 배치되는 구조는 구현이 불가능합니다.
  3. 기본적으로 엑셀 시트의 컬럼 너비를 사용하지만, 정확한 너비값을 사용하기를 원하면 컬럼 너비 추출 버튼을 클릭하면 현상태의 컬럼 너비가 너비 셀에 기록됩니다. 너비 셀 값을 수정하고 너비값 적용버튼을 클릭하면 작성한 너비값이 실제 시트상의 컬럼 너비로 적용됩니다. 너비값을 수정 후 너비값 적용을 하지 않으면 보이는 크기와 실제 그리드 생성시의 크기가 달라질 수 있습니다.
  4. 컬럼형태가 완성이 되면 코드 생성버튼을 클릭하면 상단에 소스가 생성됩니다. 소스를 복사하여 사용하면 되고 미리보기버튼을 클릭하면 하단에 리얼그리드 객체를 표시합니다. 단 인터넷 통신을 이용하여 접근하므로 외부망이 차단되어 있으면 미리보기 사용이 불가능합니다.
